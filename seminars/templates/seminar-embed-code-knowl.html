Including this HTML code on your webpage will display the series schedule there. The code consists of a <div> tag and a <script> tag.

To include the schedules of multiple series, you need the <div> tag for each but the <script> tag only once, so copy the code block for each series to your webpage and remove all but one copy of the <script> tag.

By default, our styling of the schedule overrides your webpage's styling. If you want your webpage's styling to apply, then after copying the code, change 'addCSS': true to 'addCSS': false in the <script> tag.
