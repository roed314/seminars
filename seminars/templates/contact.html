Contacting the Math Seminars developers

  • Urgent question? Send an email to the developers' mailing list.

  • Urgent error report? Send an email.

  • Other question or error report? Use the feedback form.

  • Have a suggestion for improving the site? Use the feedback form.

  • Want to help improve the site yourself? Send an email.


To be most helpful, error reports should include

  • a one line description of the problem,
  • the exact URL which shows the issue (copied and pasted from the browser location bar), and
  • a full description of the problem.

The current list of issues being tracked can be found here.
